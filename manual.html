Getting to your data

How to send queries to InfluxDb

When first visiting the website an query is automaticly sent. By default this query includes the last 48h. To select specific start- and end-dates use the given "datepicker". Click on show to send the query.

Whole days will be selected and it can take some time to get a response from the database. All time is measured in UTC! In case of trawls going over (UTC) midnight the query end at 0pm and the real end of the trawl can not be shown.

How to find a specific trawl

After the query finished the list will be filled with IDs of trawls. Each element in the list shows the start of measuring and the end. Initially all of the data is displayed and can later on still be viewed by clicking on the first element in the list, here 1. .

Clicking on a specific ID inside the list, here 2. with ID:110, will zoom into the charts. The x- and y- axis of all charts will be adjusted to that timeframe of the trawl.

How to zoom inside charts

You can zoom into a timeframe by "brushing" over a single chart. By releasing your touchscreen or click all charts will be updated to show the marked timeframe. The end of the timeframe will always be the line to the right, here end , and the line to the left marks the start.

In this example the selected time would be between 10:38 and 10:44. Zooming back out can easily be done by selecting an element inside the ID-list, explained in previous section. Brushing is only available for horizontal profiles. Should the zoom factor be too large the y-axis converts to a single-number-scale, which implies in this timeframe is no data available.

How to utilize the map

After each query the map draws lines to display all trawls in that period. By clicking on a line the trawls ID will be shown.

By clicking on an ID inside the list the map will also show up- and downcast position, which are first and last coordinates of the selected track. Blue marks the upcast and orange the downcast. If the trawl was between two days it shows the last coordinates inside the query. In addition basic feature like zoom and dragging to move the map are available.
